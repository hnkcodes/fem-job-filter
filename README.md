# Job Listings with Filtering

A solution to the Frontend Mentor **Job Listings with Filtering** challenge.

## Overview

This project displays a list of job postings and allows users to filter jobs by clicking tags such as role, language, or tools. Multiple filters can be combined, and only jobs matching all selected filters are displayed.

## Features

- Display job listings from JSON data
- Filter by:
  - Role
  - Languages
  - Tools
- Multiple filters (AND search)
- Remove individual filters
- Clear all filters
- Responsive layout

## Built with

- React
- Redux Toolkit
- CSS Grid
- Flexbox
